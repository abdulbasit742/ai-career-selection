# Catalog data

`sample-institutes.json` contains fictional demonstration records only. Names, URLs, fees, requirements, and deadlines are not real admissions information.

A production data record should include:

- an official or licensed source name and URL
- a verification date
- currency and fee period
- accreditation/status evidence
- program eligibility and deadline provenance
- an update owner and refresh schedule

Do not scrape or republish institute data without checking source terms and applicable law.
